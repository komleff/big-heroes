## 3. Компоненты UI

### 3.1. Кнопка основного действия (ButtonPrimary)

- Форма: скруглённый прямоугольник, borderRadius = 16px
- Размер: ширина 80% экрана, высота 64–72px
- Фон: линейный градиент сверху вниз accent_green → accent_green_dark
- Обводка: 3px, accent_green_dark (нижний край толще — эффект 3D)
- Тень: 0 4px 0 accent_green_dark (создаёт объём)
- Текст: белый, Bold, 28px, тень 0 2px 0 rgba(0,0,0,0.3)
- Состояние нажатия: сдвиг вниз на 2px, тень уменьшается

### 3.2. Кнопка вторичного действия (ButtonSecondary)

- Форма: скруглённый прямоугольник, borderRadius = 12px
- Размер: по контенту + padding 16px 24px
- Фон: линейный градиент accent_magenta → #B03090
- Обводка: 2px, #8B2070
- Тень: 0 3px 0 #8B2070
- Текст: белый, Bold, 18px

### 3.3. Кнопка «Отступить» (ButtonDanger)

- Форма: скруглённый прямоугольник, borderRadius = 16px
- Размер: ширина 80% экрана, высота 56px
- Фон: bg_card (#3D4670) с обводкой 2px #5A6590
- Текст: белый, Bold, 22px
- Подпись: мелкий текст под кнопкой «Билеты не вернутся»

### 3.4. Карточка игрока в списке (PlayerRow)

- Форма: скруглённый прямоугольник, borderRadius = 12px
- Размер: ширина списка − 16px отступ, высота 72px
- Фон: bg_card
- Содержимое слева направо:
  - Бейдж ранга (медаль для топ-3 / текст «# N» для остальных)
  - Аватар (40×40px, скруглённый квадрат borderRadius = 8px)
  - Имя игрока
  - Масса (число + «т» или «гр»)
  - Кнопка «Напасть» (ButtonSecondary) со стоимостью
- Строка текущего игрока: фон bg_card_highlight, обводка 2px accent_cyan

### 3.5. Экран сравнения (BattleMatchup)

- Фон: градиент от красного (#8B0000) слева к синему (#00008B) справа
- Разделитель: диагональная молния / зигзаг по центру
- Слева: карточка атакующего (аватар, масса, «атакует»)
- Справа: карточка защитника (аватар, масса, «отступает»)
- Над карточками: аватар + имя + ранг (место)
- Карточка: скруглённый квадрат 160×160px, borderRadius = 16px, белая обводка 3px
- Масса: крупный шрифт 36px Extra Bold под карточкой
- Внизу: ButtonPrimary «АТАКОВАТЬ» + ButtonDanger «ОТСТУПИТЬ»
- Между кнопками и карточками: «Награда: +500гр  Риск: высокий»

### 3.6. Экран победы (VictoryScreen)

- Фон: тёмный с ареной / рингом внизу, виньетка
- Баннер: лента (ribbon) accent_pink, текст «ПОБЕДА!» белый 42px Black
- Подпись: «Вы пробили защиту» 18px Medium
- Центр: крупное изображение яйца / героя с glow-эффектом
- Под героем: масса «500гр» accent_cyan 36px Extra Bold
- Карточки наград: два блока в ряд
  - Левый: «Награда!» + «+48 гр» + «420 г → 500гр»
  - Правый (премиум): «Премиум бонус!» + «+20 гр» (с золотой обводкой)
- Внизу: ButtonPrimary «Продолжить»

### 3.7. Нижняя навигация (BottomNav)

- Фон: bg_secondary с верхней обводкой 1px #4A5568
- 5 вкладок в ряд, каждая:
  - Иконка 32×32px (линейная, белая)
  - Подпись 12px Medium белая
- Активная вкладка: иконка accent_yellow, подпись accent_yellow

### 3.8. Полоса прогресса (ProgressBar)

- Фон: тёмный (#1A2030), borderRadius = 8px, высота 16px
- Заполнение: градиент accent_green → accent_yellow
- Обводка: 2px bg_secondary
- Текст по центру: «420 / 500» белый 12px Bold

### 3.9. Панель ресурсов (ResourceBar)

- Располагается в верхнем правом углу
- Иконка ресурса (Gold = жёлтая монета / сыр) + число
- Фон: pill-shaped bg_card, borderRadius = 20px, padding 8px 16px
- Текст: 18px Bold белый

## 4. Общие принципы

### Эффекты
- Drop-shadow на кнопках и карточках: 0 4px 8px rgba(0,0,0,0.3)
- Glow на важных элементах: box-shadow 0 0 20px accent_color (30% opacity)
- Текстовые тени на заголовках: 0 2px 4px rgba(0,0,0,0.5)
- 3D-эффект на кнопках через нижнюю тень

### Скругления
- Кнопки: 12–16px
- Карточки: 12px
- Аватары: 8px
- Прогресс-бары: 8px
- Модалки: 20px

### Отступы
- Между карточками в списке: 8px
- Padding внутри карточки: 12px 16px
- Боковые отступы экрана: 16px
- Отступ от верхнего края до заголовка: 48px (с учётом status bar)

### Анимации (для PixiJS)
- Появление экрана: fade in 300ms + scale от 0.95 до 1.0
- Нажатие кнопки: scale до 0.95 за 100ms, возврат за 100ms
- Появление награды: scale от 0 до 1.2 за 200ms, затем до 1.0 за 100ms (bounce)
- Удар в бою: shake 50ms × 3
- Полоса HP: tween ширины 300ms ease-out

### Ориентация
- Portrait (вертикальная), 390×844 (iPhone 14 / стандартный мобильный)
- Canvas PixiJS масштабируется под реальный экран с сохранением пропорций
