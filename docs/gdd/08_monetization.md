## Философия

Донат покупает удобство, скорость и доступ — не прямую силу. Платящий игрок проходит больше циклов в день и выносит больше лута, но обязан играть PvE и PvP. Основной цикл не заменяется деньгами.

Гибридная монетизация (IAP + IAA) почти обязательна для midcore-проекта в 2025–2026. Она позволяет:
- Поднимать early LTV через рекламу, пока IAP-воронка раскачивается.
- Снижать риск провала окупаемости в Tier-2/3 гео.
- Монетизировать неплатящих игроков.

## IAP (покупки в приложении)

### MVP: стартовые SKU

| SKU | Содержимое | Цена (USD) | Категория |
|-----|-----------|------------|-----------|
| starter_pack | 200 Gems + 500 Gold + 1 предмет tier 1 (случайный) | $0.99 | Стартовый бандл |
| gems_small | 100 Gems | $0.99 | Валюта |
| gems_medium | 550 Gems (бонус +10%) | $4.99 | Валюта |
| gems_large | 1200 Gems (бонус +20%) | $9.99 | Валюта |
| gold_pack | 2000 Gold | $0.99 | Валюта |
| ticket_bundle | 3 campaign_ticket + 2 arena_ticket | $1.99 | Билеты |

### Ограничения

- starter_pack: покупается 1 раз.
- Пакеты валюты: без ограничений.
- ticket_bundle: без ограничений, но не более purchase_cooldown между покупками (0 в MVP, возможно 24 часа post-MVP).

### Расширение рюкзака

Покупается за Gems внутри игры (не как IAP-SKU напрямую). Цены — см. [06_inventory](06_inventory.md).

## IAA (реклама в приложении)

### Форматы и размещение

| Формат | Размещение | Триггер | Награда | Лимит |
|--------|-----------|---------|---------|-------|
| Rewarded video | Поражение в PvE | Кнопка «Сохранить 1 предмет» | Сохранение 1 предмета из рюкзака | 1 раз за поход |
| Rewarded video | Экран ремонта | Кнопка «Скидка на ремонт» | repair_ad_discount (30–50%) | 1 раз за сессию ремонта |
| Rewarded video | Аренный сундук (при открытии) | Кнопка «Удвоить Gold из сундука» | × 2 Gold-награда | 1 раз за сундук |
| Rewarded video | Завершение PvE-похода (победа) | Кнопка «+1 бонусный предмет» | Дополнительный лут из LootSystem | 1 раз за поход |
| Interstitial | Между PvP-боями (каждый N-й бой) | Автоматически | — | 1 за 3–5 боёв, не в бою |

### Принципы размещения рекламы

- Rewarded — основной формат. Всегда по инициативе игрока.
- Interstitial — осторожно, только вне боевых моментов, не чаще 1 раз за 3–5 боёв.
- Никакой рекламы в момент боя, при выборе на развилке или при открытии сундука.
- Баннеры — не в MVP. Post-MVP: только на экранах ожидания, если unit economics требует.

### Целевые метрики IAA

| Метрика | Цель MVP |
|---------|----------|
| Доля rewarded-users среди активных | 20–30% |
| Среднее кол-во rewarded просмотров / DAU | 2–4 |
| ad_frequency без роста churn | Подтвердить в A/B |

## VIP / Подписка (post-MVP, месяц 3–4)

### Концепт

Месячная подписка, дающая QoL-бонусы. Не даёт прямой силы.

### Предполагаемые бонусы

| Бонус | Описание |
|-------|----------|
| +1 protected_slot | Дополнительный защищённый слот рюкзака |
| auto_repair_limit | Автоматический ремонт до N Gold за сессию |
| +1 free_campaign_ticket / день | Дополнительный билет кампании |
| +1 free_arena_ticket / день | Дополнительный билет арены |
| mass_gain_bonus | +10–15% к набору массы |
| gold_gain_bonus | +10–15% к набору Gold |

### Цена

$4.99–$9.99/месяц (определить через A/B).

## Season Pass (post-MVP, месяц 4–5)

### Структура

Два трека: бесплатный и премиальный.

| Уровень | Бесплатный трек | Премиальный трек |
|---------|----------------|-----------------|
| 1–10 | Gold, бустеры | + Gems, предметы tier текущей главы |
| 11–20 | Gold, бустеры, 1 предмет | + Gems, осколки героев, косметика |
| 21–30 | Gold, бустеры | + Gems, эксклюзивный предмет, рамка |

Длительность: 4–6 недель (синхронизировать с сезоном лиг).
Прогресс: через XP от PvE-походов и PvP-боёв.

### Цена

$4.99–$9.99 (определить через A/B).

## Воронка монетизации

```
Install
  │
  ▼
FTUE завершён ────────── 45–55% conversion
  │
  ▼
Engaged: 2+ PvE забега за 24ч
  │
  ├──▶ Core-loop: PvE → PvP выполнен 1 раз ── 35–45% от engaged
  │         │
  │         ├──▶ Retained D7 ─── 5–6%
  │         │
  │         ├──▶ Payer: 1-я покупка (D7) ─── 1.5–2.5%
  │         │         │
  │         │         └──▶ Repeat payer (D30) ─── 30–50% от payers
  │         │
  │         └──▶ Ad engaged: 3+ rewarded / день ─── 20–30% от active
  │
  └──▶ Churn (не замкнул цикл) ─── 55–65%
```

## Что НЕ продаётся за деньги

| Запрет | Причина |
|--------|---------|
| Прямая покупка массы | Обесценивает PvE-цикл |
| Прямая покупка рейтинга | Обесценивает PvP |
| Гарантированные предметы tier ≥ 3 | P2W-ощущение |
| Пропуск боя на арене (auto-win) | Убивает геймплей |
| Бесконечные билеты (без кулдауна) | Разрушает пейсмейкер |
