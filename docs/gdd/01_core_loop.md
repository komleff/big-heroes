## Описание шагов

1. **Подготовка.** Игрок в хабе надевает снаряжение (оружие, щит, аксессуар), заполняет пояс расходниками (доступен с уровня аккаунта 2), при наличии активирует мета-бустер. Тратит campaign_ticket.

2. **PvE-поход.** Начинается со Святилища — выбор 1 реликвии из 3 (пассивный бонус на весь поход). Далее маршрут из 8–10 узлов: бои с мобами, сундуки, магазин, лагерь, события. На развилках (3–4) выбирает путь из 2–3 вариантов, часть скрыта («???»). Перед каждым боем: HP = масса (полное восстановление), выбор расходника с пояса + команда → автобой 2–3 секунды. Победа = +масса, +шанс лута. Поражение = поход завершается.

3. **Вынос лута.** Масса, снаряжение, расходники, Gold. Лут ограничен рюкзаком (4+ слота). При победе над боссом лишнее продаётся за Gold. При поражении — лишний лут (вне рюкзака) теряется. Масса сохраняется на момент входа в проигранный бой. При добровольном выходе — вся набранная масса сохраняется.

4. **Хаб.** Возвращается, экипирует новое снаряжение, чинит старое за Gold. Оценивает готовность к арене.

5. **PvP-арена.** Доступна с уровня аккаунта 5. Тратит arena_ticket. Серия боёв против ботов. HP = масса перед каждым боем. Выбор расходника + команда → автобой. Победа: +рейтинг, +прогресс сундука, масса без изменений, −1 прочность. Поражение: −рейтинг, −10% массы, −1 прочность. Сессия до истощения прочности или массы.

6. **Награды арены.** При заполнении сундука: расходники, осколки героев, элементы коллекций, снаряжение (редко). Награды усиливают следующий PvE-поход.

7. **Возврат в PvE.** Потребность в новой массе (потеряна в PvP) и снаряжении (изношено) мотивирует новый поход. Цикл замыкается.

## Эмоциональная карта

| Этап | Эмоция | Драйвер |
|------|--------|---------|
| Подготовка | Предвкушение, контроль | Выбор билда, заполнение пояса |
| Святилище | Стратегический выбор | Реликвия определяет стиль похода |
| PvE-поход | Азарт, любопытство | Развилки, случайный лут, выбор команды |
| Вынос лута | Облегчение / досада | Рюкзак заполнен, что-то не влезло |
| PvP-арена | Доминирование, напряжение | Серия побед, истощение прочности, решение «когда выйти» |
| Награды | Удовлетворение | Прогресс сундука, новые предметы |
| Ремонт | Планирование | Что чинить, что заменить, когда идти снова |

## Временные ожидания

| Этап | Целевая длительность |
|------|---------------------|
| Подготовка | 30–60 сек |
| PvE-поход (до босса) | 5–10 мин (12 узлов × 20–40 сек) |
| PvP-сессия | 3–5 мин (5–7 боёв × 15–30 сек) |
| Ремонт и оценка | 30–60 сек |
| **Полный цикл** | **10–15 мин** |

Полных циклов в день: 4–6 (активный игрок).

## Ключевая гипотеза

Двойной цикл работает, если:

1. PvE подготавливает к PvP (масса и снаряжение критичны для арены).
2. PvP мотивирует вернуться в PvE (износ снаряжения, потеря массы, награды сундука).
3. Игрок понимает ритм в первые 48 часов (core-loop conversion ≥ 35–45%).
