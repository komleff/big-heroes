## Главный цикл

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  Подготовка  │────▶│  PvE-поход   │────▶│   Вынос      │
│  (экипировка,│     │  (бои, лут,  │     │   лута       │
│   бустеры)   │     │   развилки)  │     │  (рюкзак)    │
└──────────────┘     └──────────────┘     └──────┬───────┘
       ▲                                         │
       │                                         ▼
┌──────┴───────┐     ┌──────────────┐     ┌──────────────┐
│   Ремонт,    │◀────│   Награды    │◀────│  PvP-арена   │
│   усиление   │     │  (сундук,    │     │  (бои с      │
│              │     │   ранг)      │     │   ботами)    │
└──────────────┘     └──────────────┘     └──────────────┘
```

## Описание шагов

1. **Подготовка.** Игрок в хабе выбирает снаряжение из инвентаря (weapon, armor, accessory), выбирает PvE-бустеры. Тратит campaign_ticket. Подтверждает вход.

2. **PvE-поход.** Проходит маршрут из 8–10 узлов: бои с мобами, сундуки, магазин, лагерь, события. На развилках (3–4 за поход) выбирает путь из 2–3 вариантов.

3. **Вынос лута.** Набирает массу, снаряжение, бустеры, Gold. Лут ограничен рюкзаком (4 слота). При заполнении — решает что оставить, что выбросить. При победе над боссом лишнее продаётся за Gold. При поражении лишнее теряется.

4. **Хаб.** Возвращается, экипирует новое снаряжение, чинит старое. Оценивает готовность к арене.

5. **PvP-арена.** Тратит arena_ticket, выбирает противников из списка ботов, бьётся до истощения (масса, прочность, собственное решение выйти).

6. **Результат PvP.** Победы повышают рейтинг, продвигают аренный сундук. Поражения снижают рейтинг и массу. Каждый бой изнашивает снаряжение.

7. **Награды.** При заполнении аренного сундука — PvE-бустеры, осколки героев, элементы коллекций.

8. **Возврат в PvE.** Награды арены + потребность в новой массе и снаряжении мотивируют новый поход.

## Эмоциональная карта

| Этап | Эмоция | Драйвер |
|------|--------|---------|
| Подготовка | Предвкушение, контроль | Выбор билда, оптимизация |
| PvE-поход | Азарт, любопытство | Развилки, случайный лут, риск |
| Вынос лута | Облегчение / досада | Рюкзак заполнен, что-то не влезло |
| PvP-арена | Доминирование, напряжение | Серия побед, истощение ресурсов |
| Награды | Удовлетворение | Прогресс сундука, новые предметы |
| Ремонт | Планирование | Решение что чинить, что заменить |

## Временные ожидания

| Этап | Целевая длительность | Примечание |
|------|---------------------|------------|
| Подготовка | 30–60 сек | Быстрый выбор, не должно тормозить |
| PvE-поход (полный, до босса) | 5–8 мин | 8–10 узлов × 20–40 сек каждый |
| PvP-сессия (серия боёв) | 3–5 мин | 4–8 боёв × 20–30 сек |
| Ремонт и оценка | 30–60 сек | Между контурами |
| **Полный цикл** | **10–15 мин** | Один оборот PvE → PvP → награды |

Целевое количество полных циклов в день: 4–6 (для активного игрока).

## Ключевая гипотеза

Двойной цикл работает, если выполнены три условия:

1. PvE действительно подготавливает к PvP (масса и снаряжение из PvE критичны для результата на арене).
2. PvP действительно мотивирует возвращаться в PvE (износ снаряжения, потеря массы, награды сундука усиливают PvE).
3. Игрок понимает ритм и принимает его в первые 48 часов (core-loop conversion ≥ 35–45%).

Эти условия — основные критерии валидации MVP.
