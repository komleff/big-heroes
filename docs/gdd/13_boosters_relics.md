## Место в системе

Четыре слоя предметов, каждый с уникальной ролью:

| Слой | Длительность | Решение | Пример |
|------|-------------|---------|--------|
| Снаряжение | Пока не сломается | До похода | Меч +3 силы |
| Расходники | 1 бой / 1 раз | Перед боем или между боями | Зелье силы +8 |
| Бустеры | N боёв + M минут | До начала сессии | ×2 Gold на 5 боёв / 30 мин |
| Реликвии | Весь поход | Автоматически при находке | +20% массы из побед |

Бустеры и реликвии действуют фоном, не занимают пояс, не требуют клика перед боем.

## Бустеры

### Определение

Мета-усилитель, действующий определённое время и/или число боёв. Активируется в хабе до начала похода или аренной сессии. Усиливает фарм и мета-прогрессию, не боевые параметры. Без бустера играть можно, с бустером — выгоднее.

### Ограничения

**Двойное ограничение:** N боёв + M минут. Что наступит раньше — завершает бустер.

**Стакание:** одновременно активен максимум 1 бустер каждой категории. Разные категории стакаются.

| Категория | Максимум одновременно |
|-----------|-----------------------|
| Фарм (PvE) | 1 |
| PvP (арена) | 1 |
| Камбек | 1 |
| **Итого** | **до 3 одновременно** |

**Срок годности:** бустер истекает через 24–48 часов после получения, если не активирован.

**Область действия Gold-бустера:** умножает только Gold из игровых источников. Gold, купленный за Gems или реальные деньги, НЕ умножается.

### Активация

На экране хаба перед нажатием «Поход» или «Арена»:
1. Кнопка «Бустеры» → список доступных.
2. Выбрать → подтвердить.
3. Иконка бустера появляется в ResourceBar с обратным отсчётом.

**Напоминание:** при входе в поход/арену, если есть неактивный бустер — подсказка.

### Визуализация

```
[Gold: 1500] [Масса: 85 кг] [⚡×2 Gold: 4/8 боёв, 22:15]
```

### Камбек-бонусы

Выдаются автоматически при условии. Не покупаются за валюту.

| ID | Название | Условие | Эффект | Боёв | Минут |
|----|----------|---------|--------|------|-------|
| comeback_return | Бонус возвращения | Не заходил 24+ часов | ×2 масса + ×2 Gold | 5 | 30 |
| comeback_defeat | Бонус реванша | 3 поражения подряд в PvP | Щит рейтинга + щит массы | 3 | 20 |
| comeback_arena | Бонус арены | Вылет из лиги (сезонный сброс) | ×2 сундук + ×1.5 масса | 5 | 30 |

### Цикл камбек-бонуса

```
Получить камбек-бонус (×2 масса)
  → Быстро набрать массу в PvE
    → Выйти на арену с преимуществом
      → Подняться в рейтинге
        → Противники сильнее → возможен вылет
          → При вылете → новый камбек-бонус
```

## Реликвии

### Определение

Пассивный бонус, найденный в PvE-походе и действующий до конца этого похода. Не переносится между походами. Рогалик-элемент, адаптированный к короткому формату Big Heroes.

### Отличие от других слоёв

- Не занимает слот (ни экипировки, ни пояса, ни рюкзака).
- Не ломается, не тратится.
- Работает автоматически, без клика.
- Исчезает по окончании похода.

### Гарантированные узлы реликвий

| Узел | Тип | Описание |
|------|-----|----------|
| Узел 1 | Святилище | Гарантированный. Выбор 1 из 3 реликвий перед первым боем. |
| Узел 5–6 | Древний сундук | Гарантированный. Ещё 1 реликвия (случайная или выбор 1 из 2). |
| Элита (при победе) | Шанс | 30–50% шанс реликвии дополнительно к обычному луту. |
| Босс (при победе) | Гарантированный | Выбор 1 из 3 реликвий как часть награды. |

Итого за типичный поход: 2 гарантированные + 0–2 дополнительные.

### Лимит

max_relics = 3. При превышении — выбрать какую заменить.

### Визуализация

```
[Масса: 88 кг] [Gold: 320] [🔮🛡️⭐]  ← 3 реликвии
```

## Pity-система лута

| Параметр | Значение |
|----------|----------|
| pity_counter | 5 |
| Эффект | Каждый 5-й предмет из сундуков и боёв — гарантированный tier+1 |
| Сброс | Счётчик сбрасывается при срабатывании |
| Область | В рамках одного похода |
| С бустером meta_loot_pity | pity_counter = 3 (каждый 3-й) |

Счётчик отображается в интерфейсе рюкзака: «До улучшенного лута: 2/5».
