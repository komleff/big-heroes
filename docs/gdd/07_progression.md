## Общий принцип

PvE-кампания и PvP-арена делятся на уровни ценности (главы и лиги). Прогресс в одном направлении требует успеха в другом. Это связывает два контура в единый маршрут развития.

## Главы PvE

| Глава | Tier | Название | Мин. масса входа | Условие разблокировки |
|-------|------|----------|------------------|----------------------|
| 1 | 1 (Обычная) | Первый рубеж | 0 | Стартовая |
| 2 | 2 (Необычная) | Лесная чаща | 200–400 | Пройти главу 1 + Лига 2 |
| 3 | 3 (Редкая) | Горные пещеры | 600–1000 | Пройти главу 2 + Лига 3 |
| 4 | 4 (Эпическая) | Тёмная крепость | 1500–2500 | Пройти главу 3 + Лига 4 |
| 5 | 5 (Легендарная) | Вершина мира | 4000–6000 | Пройти главу 4 + Лига 5 |

MVP включает только главу 1. Главы 2–5 — post-MVP контент.

### Что даёт более высокая глава

| Аспект | Разница |
|--------|---------|
| Снаряжение | Предметы более высокого tier (выше модификаторы) |
| Масса | Больше mass_per_combat / elite / boss |
| Gold | Больше gold_per_combat / elite / boss |
| Мобы | Сильнее (chapter_base_hp и chapter_base_attack масштабируются) |
| Бустеры | Более мощные PvE и PvP бустеры |

### Масштабирование мобов по главам

```
chapter_base_hp(chapter) = base_hp_ch1 × chapter_scaling_hp ^ (chapter − 1)
chapter_base_attack(chapter) = base_attack_ch1 × chapter_scaling_attack ^ (chapter − 1)
```

| Параметр | Диапазон |
|----------|----------|
| base_hp_ch1 | 30–50 |
| base_attack_ch1 | 5–10 |
| chapter_scaling_hp | 1.8–2.5 |
| chapter_scaling_attack | 1.5–2.0 |

## Лиги PvP

| Лига | Номер | Мин. рейтинг | Мин. масса | Условие разблокировки |
|------|-------|-------------|-----------|----------------------|
| Бронзовая | 1 | 0 | 0 | Стартовая |
| Серебряная | 2 | 1200–1500 | 200–400 | Автоматически при достижении рейтинга |
| Золотая | 3 | 1800–2200 | 600–1000 | Автоматически |
| Платиновая | 4 | 2500–3000 | 1500–2500 | Автоматически |
| Бриллиантовая | 5 | 3500–4000 | 4000–6000 | Автоматически |

MVP включает только лигу 1. Лиги 2–5 — post-MVP.

### Что даёт более высокая лига

| Аспект | Разница |
|--------|---------|
| Аренный сундук | Лучше награды (более высокий tier предметов и бустеров) |
| Боты | Сильнее (масштабируются под лигу) |
| Осколки героев | Выше шанс drop из сундука лиги |
| Коллекции | Уникальные элементы коллекций для каждой лиги |

### Связь глав и лиг

Условие открытия следующей главы — пройти текущую главу **И** достичь нужной лиги. Условие достижения лиги — набрать рейтинг, что требует массу и снаряжение из PvE.

```
Глава 1 + Лига 1 ──▶ Глава 2 + Лига 2 ──▶ Глава 3 + Лига 3 ──▶ ...
    │                     │                     │
    │ PvE готовит массу   │ PvP требует массу   │
    │ и снаряжение        │ для следующей лиги   │
    └────── PvP ◀─────────┘                     │
                          └────── PvP ◀─────────┘
```

## Прогрессия «пройти главу»

Глава считается пройденной, когда игрок:
1. Победил босса главы хотя бы 1 раз.
2. Статус chapter_completed[chapter_id] = true сохраняется перманентно.

Повторное прохождение пройденной главы разрешено (для фарма).

---

## Post-MVP: сезоны

С месяца 4–5 лиги получают сезонный сброс:
- Рейтинг частично сбрасывается (new_rating = rating × season_reset_ratio, где season_reset_ratio = 0.5–0.7).
- Аренный сундук сбрасывается.
- Сезонные награды за финальный ранг.

Длительность сезона: 4–6 недель.

---

## Edge cases

- Игрок достиг рейтинга лиги 2, но не прошёл главу 1: лига 2 разблокирована, глава 2 — нет. Игрок может бороться в лиге 2, но не может входить в главу 2.
- Игрок прошёл главу 1, но рейтинг упал ниже порога лиги 2: рейтинг не откатывает лигу. Достигнутая лига сохраняется (no demotion в MVP). Post-MVP: возможен дефолтный сброс в начале сезона.
- Игрок хочет повторно пройти главу 1 после разблокировки главы 2: разрешено. Лут из главы 1 будет tier 1.
