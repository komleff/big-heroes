## Экипировка (3 слота)

| Слот | Основной параметр | Усиливает | Ломается при |
|------|----|-----------|-------------|
| Оружие | +Сила | Атаку | cmd_attack |
| Щит/Броня | +Броня (+HP) | Блок (выравниватель) | cmd_block |
| Аксессуар | +Удача | Своё умение | Использовании умения |

---

## Правило износа

**«Ломается тот предмет, чья команда использована.»**

Предмет, чья команда не выбрана, не теряет прочность.

### Прочность по tier

| Tier | Редкость | Прочность |
|------|----------|-----------|
| 1 | Обычный | 3 |
| 2 | Необычный | 5 |
| 3 | Редкий | 7 |
| 4 | Эпический | 9 |

---

## Состояния предмета

### Прочность > 0 (целый или частично изношен)

Все параметры работают полностью. Нет штрафов. Прочность 1 = прочность 3 по бонусам.

### Прочность = 0 (сломан)

| Слот | Параметры | Команда |
|------|-----------|---------|
| Оружие | +Сила = 0 (голые руки) | Атака **доступна** (без бонуса) |
| Щит | +Броня = 0 | Блок **доступен** (слабый, block_power = 0.3) |
| Аксессуар | +Удача = 0 | Команда **заблокирована** |

Сломанный предмет визуально потускнел + красная метка. Использование команды от сломанного оружия/щита не снижает прочность дальше (уже 0).

## Пояс

| Аспект | Правило |
|--------|---------|
| Размер | 2 (ур. аккаунта 2) → 3 (ур. 7) → 4 (ур. 12) |
| Содержимое | Любые расходники |
| Перед боем | Боевые расходники, аптечка |
| На перекрёстке | Разведывательные расходники |
| Между боями | Всё (использовать, переложить из рюкзака) |
| Лимит за бой | 1 боевой расходник |
| Лимит за перекрёсток | 1 разведывательный |

## Рюкзак

| Аспект | Правило |
|--------|---------|
| Базовый размер | 4 слота |
| Максимум | 8 (расширение за Gems) |
| Котомка | +1/2/3 временных слота на поход |
| Защищённый слот | 1 (+1 VIP). Предмет не теряется при поражении |
| Перед боем | **Недоступен** |
| Между боями | Доступен |
| При завершении похода | Содержимое → инвентарь |

### Расширение

```
expand_cost(N) = base_expand_cost × expand_multiplier ^ N
```

Конкретные цены: ⏳ Этап балансировки.

## Ремонт

### В хабе (полный ремонт)

Полностью восстанавливает прочность. Оплачивается Gold.

```
repair_cost = (max_durability − current_durability) × cost_per_point × tier
```

Конкретные значения cost_per_point: ⏳ Этап балансировки.

### В походе (ремкомплект)

| Tier | +Прочность |
|------|-----------|
| 1 | +1 |
| 2 | +2 |
| 3 | +3 |
| 4 | +4 |

Прочность не превышает максимум. Используется между боями.

---

## Выход из похода

Кнопка «Выйти из похода» в каждом узле (кроме экрана предбоя).

| Аспект | Правило |
|--------|---------|
| Рюкзак | → инвентарь |
| Лишний лут | Теряется |
| Масса | Сохраняется масса, с которой вошёл в последний бой |

## Edge cases

- Запас полон, снимаешь экипировку: «Запас полон. Продайте что-нибудь».
- Рюкзак полон, новый лут: экран выбора — выбросить или отказаться.
- Все предметы сломаны: бой разрешён. Атака и блок без бонусов. Предупреждение.
- Ремкомплект на предмет с полной прочностью: кнопка неактивна.
- Сломанный аксессуар: умение заблокировано. Иконка серая + замок.
