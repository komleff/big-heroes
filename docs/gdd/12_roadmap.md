## Стратегический план на 6 месяцев

### Месяц 1 — Прототип core feel

**Продуктовая цель:** Играбельный PvE-забег с боями, лутом и рюкзаком.

| Задача | Приоритет |
|--------|-----------|
| PvE-забег: карта, узлы, развилки | P0 |
| Боевая система: пошаговый бой | P0 |
| Рюкзак и ограничения выноса | P0 |
| Прочность снаряжения (подготовка, без расхода) | P1 |
| Ремонт (базовый) | P1 |
| Аналитика v0: ключевые события | P1 |
| Dev-панель (базовая) | P2 |

**Метрики успеха:**
- FTUE completion ≥ 45–55%.
- 1-й PvE-run completion ≥ 60%.

**Ресурсы:** 1 продакт/дизайнер, 2–3 client, 1 server, 1 аналитик (part-time), 1 QA.

---

### Месяц 2 — Замкнуть цикл PvE → PvP

**Продуктовая цель:** Работающий двойной цикл. Арена с ботами, сундук, рейтинг.

| Задача | Приоритет |
|--------|-----------|
| Async PvP: BotFactory, PvpSystem, PvpLobbyScene | P0 |
| Matchmaking: подбор ботов по рейтингу и массе | P0 |
| Аренный сундук: шкала, награды | P0 |
| Износ снаряжения в PvP | P0 |
| Потеря массы при поражении | P0 |
| Баланс массы: sources vs sinks | P1 |
| Оффлайн-поражения (симуляция) | P2 |
| Итерация FTUE по данным месяца 1 | P1 |

**Метрики успеха:**
- Core-loop conversion (PvE → PvP за 48 ч) ≥ 35–45%.
- D1 ≥ 28–30%.

**Ресурсы:** +1 геймдизайнер/экономист, +1 backend.

---

### Месяц 3 — Soft launch ready

**Продуктовая цель:** Готовность к soft launch: монетизация, daily/weekly, атрибуция, минимальный live ops.

| Задача | Приоритет |
|--------|-----------|
| IAP: 3–5 SKU (стартовый бандл, Gems-пакеты, билеты) | P0 |
| IAA: rewarded placements (поражение PvE, ремонт, сундук) | P0 |
| Daily rewards (ежедневный вход) | P0 |
| Weekly challenge (еженедельное задание) | P1 |
| 1 событие (простое, 3–5 дней) | P1 |
| Простая витрина магазина | P1 |
| Антифрод (базовый) | P1 |
| Атрибуция: Adjust / AppsFlyer | P0 |
| ASO: скриншоты, описание, ключевые слова | P1 |
| A/B: FTUE длина, потеря лута severity | P2 |

**Метрики успеха:**
- D7 ≥ 5–6%.
- Устойчивый ARPDAU (измерить baseline).
- Тестируемый CPI в Tier-2 гео.

**Ресурсы:** +1 UA/ASO-специалист, +1 BI/аналитик, monetization designer.

### Месяц 4 — Контент и retention

**Продуктовая цель:** Первое расширение контента, усиление retention-петель.

| Задача | Приоритет |
|--------|-----------|
| Глава 2 (Необычная): новая зона, новые мобы, босс | P0 |
| Снаряжение tier 2: расширение пула (6–9 предметов) | P0 |
| Лига 2 (Серебряная): автоматическое продвижение | P0 |
| События v1: 2 типа событий (ротация) | P1 |
| Retention-петли: streak rewards, comeback rewards | P1 |
| Улучшение matchmaking по данным soft launch | P1 |
| A/B: rewarded placements, durability_cost | P2 |

**Метрики успеха:**
- D30 ≥ 2%.
- Рост time spent.
- Conversion в главу 2 (из игроков, прошедших главу 1).

### Месяц 5 — Season Pass и социал

**Продуктовая цель:** Season Pass как стабилизатор экономики. Скелет социальных фичей.

| Задача | Приоритет |
|--------|-----------|
| Season Pass: 2 трека, 20–30 уровней, XP-прогрессия | P0 |
| Сезонный сброс лиг | P0 |
| Клан/гильдия: скелет (создание, вступление, чат) | P1 |
| VIP-подписка: тестирование 1 варианта | P1 |
| Масштабирование UA: тесты в Tier-1 гео | P0 |
| Глубокая аналитика LTV по когортам | P0 |

**Метрики успеха:**
- Season Pass conversion ≥ 3–5% от MAU.
- D30 ≥ 2.5–3%.
- Payback test в Tier-2 пройден.

### Месяц 6 — Масштабирование

**Продуктовая цель:** Готовность к Tier-1. Второе расширение контента. Оптимизация монетизации.

| Задача | Приоритет |
|--------|-----------|
| Глава 3 (Редкая): новая зона, tier 3 снаряжение | P0 |
| Лига 3 (Золотая) | P0 |
| Оптимизация IAP SKU по данным 3 месяцев | P0 |
| Кланы v2: совместные рейды (простые, если клан скелет работает) | P1 |
| Второй герой (разблокировка через осколки) | P1 |
| Tier-1 UA scaling | P0 |
| Readiness review: готовность к полному масштабу | P0 |

**Метрики успеха:**
- CPI в Tier-1 ≤ target.
- D180 LTV ≥ CPI × 1.3 (прогноз).
- Стабильная экономика без инфляции.

## Долгосрочное видение (6–12 месяцев)

Направления, которые определяются по результатам первых 6 месяцев:

| Направление | Условие запуска | Ожидаемый эффект |
|-------------|----------------|-----------------|
| Главы 4–5 (эпическая, легендарная) | D30 ≥ 3%, стабильная экономика | Глубина прогрессии для хардкор-сегмента |
| 3–5 героев с уникальными навыками | Core-loop proven, retention стабилен | Коллекционная мета, повторяемость |
| Гильдейские рейды | Клан-скелет работает, социальные метрики растут | Социальная привязка, D60+ retention |
| Реал-тайм PvP (опционально) | Async PvP saturated, аудитория просит | Новый контур, но высокий риск и стоимость |
| Крафт и аффиксы снаряжения | Экономика стабильна, sink работает | Глубина билдинга, engagement |
| Косметика (скины, рамки, эмоции) | Payer base стабилен | Неинфляционная монетизация |

## Риски

| Риск | Влияние | Вероятность | Митигация |
|------|---------|-------------|-----------|
| Экстракшн-механика фрустрирует casual | D1/D7 ниже цели | Средняя | Страховки FTUE, protected slots, мягкие штрафы, A/B |
| Ремонт воспринимается как налог | NPS и D30 ниже цели | Средняя | Предсказуемость, rewarded-скидка, баланс Gold earn vs spend |
| Двойной цикл непонятен новичку | Core-loop conversion < 25% | Средняя | Итерации FTUE, принудительный первый PvP в tutorial |
| PvP с ботами быстро приедается | D30 retention падает | Высокая | Вариативность ботов, сезоны, события, социал (кланы) |
| Инфляция массы | Обесценивание PvP, бессмысленность стоков | Средняя | Мониторинг net mass / DAU, adjustment коэффициентов |
| CPI в Tier-1 не окупается | Невозможность масштабирования | Средняя | Начинать с Tier-2/3, доказать LTV, гибридная модель |
