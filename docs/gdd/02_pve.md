# 02 — PvE-кампания

## Роль системы

PvE-кампания — источник ресурсов и подготовка к арене. Контролируемый контур прогрессии с вариативностью и риском. Игрок проходит поход, набирает массу, добывает снаряжение и валюту, затем конвертирует результат в PvP.

---

## Вход в поход

### Условия

- 1 × campaign_ticket.
- Герой доступен.

### Подготовка перед стартом

1. Выбор снаряжения из инвентаря (weapon, shield, accessory).
2. Заполнение пояса расходниками из запаса (2–4 слота, доступен с уровня аккаунта 2).
3. Подтверждение → списание ticket → генерация маршрута.

---

## Структура похода

Поход — линейная цепочка узлов с развилками. Игрок не видит всю карту, только текущий узел и доступные пути.

### Параметры главы

| Параметр | Глава 1 (MVP) | Диапазон тюнинга (все главы) |
|----------|---------------|------------------------------|
| total_nodes | 8–10 | 6–14 |
| fork_count | 3–4 | 2–6 |
| paths_per_fork | 2–3 | 2–3 |
| boss_node | последний | всегда последний |

## Навигация по карте

### Перекрёсток (развилка)

На перекрёстке игрок видит 2–3 пути. Каждый путь показывает тип узла — или «???» (скрытый).

```
         ┌─── [⚔️ Обычный бой]
         │
[Текущий]├─── [💀 Элитный бой]
         │
         └─── [??? Неизвестно]
```

Действия игрока на перекрёстке:

| Действие | Описание |
|----------|----------|
| Выбрать путь | Тап по узлу → переход к этому узлу |
| Использовать разведывательный расходник | Факел, подзорная труба, компас и т.д. (1 за перекрёсток) |
| Использовать походный расходник | Ремкомплект из рюкзака или пояса |
| Переложить расходник из рюкзака на пояс | Подготовка к следующему бою |
| Выйти из похода | Вернуться в хаб с текущей добычей |

### Узел боя

При входе в узел с боем — экран предбоя (см. [03_battle](03_battle.md)). HP = масса (полное восстановление). Игрок видит противника и выбирает действие.

Команды, влияющие на навигацию:

| Команда | Эффект на карте |
|---------|----------------|
| Атака | Бой. При победе — остаёшься на узле, получаешь награду, идёшь дальше. При поражении — поход завершается. |
| Блок | Бой с усиленной защитой. Те же последствия. В PvE возможна ничья → тот же узел, можно выбрать другую команду. |
| Фортуна | Случайный бой + бонус удачи. Те же последствия. |
| Отступление | **Назад на перекрёсток.** Путь к этому врагу остаётся открытым. Можно перевооружиться и попробовать снова или выбрать другой путь. |
| Обход | **Вперёд мимо врага.** Противник остаётся позади, вернуться нельзя. Ни лута, ни массы. |
| Полиморф | Частичная победа. Лут и Gold как при победе, но массы не получает. |

### Отступление → возврат на перекрёсток

```
[Перекрёсток] ──▶ [Элитный моб] ──(Отступление)──▶ [Перекрёсток]
                                                      │
                                          Варианты:   │
                                          • Попробовать снова (перевооружившись)
                                          • Выбрать другой путь
                                          • Использовать расходник и вернуться
```

### Обход → проход вперёд

```
[Перекрёсток] ──▶ [Элитный моб] ──(Обход)──▶ [Следующий узел]
                        ↑
                  (заблокирован,
                   вернуться нельзя)
```

## Типы узлов

| Тип | ID | Описание | Позиция |
|-----|----|----------|---------|
| Святилище | sanctuary | Выбор 1 из 3 реликвий | Узел 1 (фиксирован) |
| Обычный бой | combat | Бой с рядовым мобом. Масса + шанс лута | 35–45% от остальных |
| Элитный бой | elite | Усиленный моб. Лучше награды + шанс реликвии | 10–15% |
| Магазин | shop | Покупка предметов и расходников за Gold | 10–15% |
| Лагерь | camp | Ремонт снаряжения или тренировка | 10–15% |
| Событие | event | Выбор из 2–3 вариантов | 5–10% |
| Сундук | chest | Лут без боя | 10–15% |
| Древний сундук | ancient_chest | Реликвия + лут | Узел 5–6 (фиксирован) |
| Босс | boss | Финальный бой. Реликвия за победу | Последний (фиксирован) |

### Узел: sanctuary (узел 1 — фиксированный)

- Перед первым боем. Игрок выбирает 1 реликвию из 3 предложенных.
- Реликвия действует весь оставшийся поход (см. [13_boosters_relics](13_boosters_relics.md)).
- Бонус реликвии отображается в верхней панели.
- Не содержит боя, лута или Gold.

### Узел: combat

- HP = масса перед боем (полное восстановление).
- При победе: масса (глава 1: 6–8 кг) + шанс лута (combat_loot_chance: 15–25%).
- При поражении: поход завершается.

### Узел: elite

- Усиленный моб (кастомные статы, см. [11_content](11_content.md)).
- При победе: масса (глава 1: 12–15 кг) + гарантированный лут + 30–50% шанс реликвии.
- Иконка красная / с черепом.

### Узел: shop

- 3–4 предмета из пула главы + расходники + полный ремонт.
- Цены: base_price × shop_price_multiplier.
- Ремонт дороже, чем в хабе: × shop_repair_multiplier (1.5–2.0).

### Узел: camp

Передышка с выбором действия:

| Вариант | Эффект |
|---------|--------|
| Починить | +1 прочность к выбранному предмету (бесплатно) |
| Тренироваться | +3–5 кг массы |
| Пересобрать пояс | Переложить расходники между рюкзаком и поясом |

Только одно действие за лагерь (кроме пересборки — доступна всегда).

### Узел: event

Случайное событие с выбором (см. таблицу в [11_content](11_content.md)).

### Узел: chest

- Лут без боя. Содержимое по LootSystem.
- Pity-система: каждый 5-й предмет из сундуков и боёв гарантированно tier+1 (см. [13_boosters_relics](13_boosters_relics.md)).

### Узел: ancient_chest (узел 5–6 — фиксированный)

- Гарантированная реликвия (случайная или выбор 1 из 2) + обычный лут как из chest.
- Визуально отличается от обычного сундука (золотое свечение).

### Узел: boss

- Финальный бой (кастомные статы, см. [11_content](11_content.md)).
- Отступление и обход **запрещены**. Кнопки скрыты.
- Гарантированные награды: масса + 1–2 предмета + Gold + реликвия (выбор 1 из 3).

## Генерация маршрута

1. Взять total_nodes из PveConfig.
2. Зафиксировать: узел 1 = sanctuary, узел 5–6 = ancient_chest, последний = boss.
3. Распределить fork_count развилок по оставшимся позициям.
4. Для каждой развилки: paths_per_fork вариантов по таблице частот.
5. Линейные узлы — тип по взвешенному случайному выбору.
6. Часть путей на развилке может быть скрыта («???»).
7. Ограничения: не более 2 боёв подряд; не более 1 магазина; минимум 1 лагерь до босса.

### Скрытые пути

На некоторых развилках часть путей показана как «???». Факел раскрывает тип. Без факела — слепой выбор.

| Шанс скрытого пути | Диапазон |
|--------------------|----------|
| hidden_path_chance | 20–40% путей на развилках |

## Правила выхода из похода

### Варианты завершения

| Условие | Рюкзак | Лишний лут | Масса |
|---------|--------|-----------|-------|
| Победа над боссом | → инвентарь | Автопродажа за Gold | Вся набранная сохраняется |

При победе над боссом — дополнительный экран «Сохрани реликвию для арены»: выбор 1 из имеющихся реликвий (см. [13_boosters_relics](13_boosters_relics.md)). Можно пропустить.
| Поражение в бою (HP = 0) | → инвентарь | Теряется | Масса на момент входа в проигранный бой |
| Добровольный выход | → инвентарь | Теряется | Вся набранная сохраняется |

### Добровольный выход

Доступен в каждом узле (кроме экрана предбоя). Кнопка «Выйти из похода». Масса сохраняется полностью.

---

## Страховки от фрустрации

| Механика | Описание | Условие |
|----------|----------|---------|
| Безопасные походы | Поражение не вызывает потери лута из рюкзака | Первые 2–3 похода |
| Защищённый слот | 1 слот рюкзака — предмет не теряется никогда | Всегда |
| VIP-защита | +1 защищённый слот | VIP-статус |
| Rewarded-страховка | Реклама → сохранить 1 дополнительный предмет | При поражении, 1 раз за поход |

---

## Edge cases

- Рюкзак полон, новый лут: экран «Рюкзак полон» → выбросить или отказаться.
- Нет campaign_ticket: предложить купить за Gems.
- Закрыл приложение во время похода: при возвращении — восстановить состояние из SaveManager.
- Отступление от обычного моба: разрешено (назад на перекрёсток).
- Обход босса: запрещено (кнопка скрыта).
- Котомка использована, рюкзак расширен: временные слоты работают до конца похода.
- Компас перед боссом: не работает (нет безопасного пути в обход босса).
