## Принятые решения

### R1. Масса при поражении/выходе из PvE

Единое правило: поражение = масса на момент входа в бой. Добровольный выход = вся масса. Множитель defeat_mass_retain_ratio убран.

Обновлены: 01, 02, 05, 06.

### R2. HP = масса, полное восстановление перед каждым боем

Единая модель для PvE и PvP. Бой — чистое решение «команда + расходник vs враг». Прочность снаряжения — единственный ресурс на истощение в походе.

Следствия:
- Аптечки heal_t1–t4 удалены (4 расходника из каталога).
- Реликвии relic_heal_after → relic_mass_on_win (+15% массы за победу).
- Реликвия relic_heal_camp → relic_camp_repair (лагерь чинит +2 вместо +1).
- Реликвия relic_full_heal_boss → relic_boss_armor (+5 брони vs босс).
- Добавлена relic_thick_skin (−15% силы врага).
- Лагерь = починить 1 предмет (+1 прочность бесплатно) ИЛИ тренировка (+3–5 кг массы) + пересборка пояса.
- Событие evt_fountain: «+25% HP» → «+10 кг массы» или «+1 прочность».
- Событие evt_trap: вариант B убрана потеря HP, заменено на потерю предмета.
- Событие evt_riddle: вариант B убрана потеря HP, заменено на потерю предмета из рюкзака.
- Формула damage_taken убрана из 03_battle.

Обновлены: 01, 02, 03, 09, 11, 13.
