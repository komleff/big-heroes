## Описание экранов

### MainMenuScene

Точка входа в игру. «Играть» / «Продолжить» → HubScene.

---

### HubScene

Центральный хаб. Референс: hub_scene_v7.html.

| Зона | Содержимое |
|------|-----------|
| Верхняя панель | Gold, Gems, билеты |
| Центр | Аватар героя, масса (кг, accent_cyan), рейтинг, уровень героя |
| Экипировка | 3 слота с пипсами прочности |
| Кнопки-действия | Квадратные 56×56: ремонт, бустеры, пояс |
| CTA | «ПОХОД» (зелёный), «АРЕНА» (magenta) с количеством билетов |
| Нижнее меню | 5 вкладок: Герой, Инвентарь, Магазин, Путь, Настройки |

---

### SanctuaryScene

Первый узел каждого PvE-похода. 3 карточки реликвий. Выбор одной → подтверждение → PveMapScene.

---

### PveMapScene

Экран текущего похода.

**Верхняя панель:** Масса (кг), Gold, мини-иконки реликвий (до 3), номер узла (N/total).

**Центр (зависит от типа узла):**
- Развилка: 2–3 карточки путей (тип + иконка или «???»).
- Линейный узел: одна карточка с кнопкой «Продолжить».

**Нижняя зона:** Мини-рюкзак, кнопка «Выйти из похода» (ButtonDanger).

---

### CampScene

Передышка в PvE-походе. Без боя.

| Элемент | Описание |
|---------|----------|
| Вариант «Починить» | Выбрать 1 предмет → +1 прочность (бесплатно) |
| Вариант «Тренироваться» | +3–5 кг массы |
| Пересборка пояса | Переложить расходники рюкзак ↔ пояс (доступно всегда) |
| Кнопка «Продолжить» | Перейти к следующему узлу |

Одно основное действие за лагерь (починить ИЛИ тренироваться). Пересборка — свободная.

---

### PreBattleScene (PvE и PvP)

Экран выбора перед боем. Референс: prebattle_scene_v9.html.

| Зона | Содержимое |
|------|-----------|
| Левая сторона | Герой: аватар, масса, сила, броня |
| Правая сторона | Противник: аватар/иконка, масса, сила, броня |
| Центр | Индикация шансов: base_win_chance (цветовая шкала) |
| Пояс | Слоты расходников. Тап → применить (1 за бой) |
| Команды | 6 кнопок с шансами рядом |
| Заблокированные | Серые + замок (аксессуар сломан) |
| Запрещённые | Скрыты (отступление/обход vs босс) |

---

### BattleScene

Анимация автобоя 2.5–3.0 секунды. Референс: battle_scene_v1.html.

Два персонажа, HP-бары, числа урона (белые/жёлтые/красные). Результат определён до анимации.

---

### LootScene

Добыча после боя / сундука в PvE. Ресурсы (+масса, +Gold) + карточка предмета. «Взять в рюкзак» / «Пропустить». Мини-рюкзак внизу.

---

### PvpLobbyScene

Список 5 ботов: имя, масса, рейтинг, уровень, пассивка, индикация силы (🟢🟡🔴). «НАПАСТЬ» на каждом, «ЗАВЕРШИТЬ СЕССИЮ» внизу. Шкала сундука вверху.

---

### PvpResultScene

| При победе | При поражении |
|-----------|---------------|
| Баннер «ПОБЕДА!» (accent_pink) | Баннер «ПОРАЖЕНИЕ» (accent_red) |
| Рейтинг +N | Рейтинг −N |
| Масса без изменений | Масса −10% |
| Прочность −1 | Прочность −1 |
| Прогресс сундука +2–3 | Прогресс сундука +0–1 |

Кнопки: «Продолжить» → PvpLobbyScene, «Завершить» → HubScene. Если сундук полон → анимация → награды.

---

### PveResultScene

**Победа над боссом:** баннер, масса (было → стало), Gold, список лута, кнопка «В хаб», rewarded (+1 предмет).

**Поражение:** баннер, масса на момент входа в бой, рюкзак сохранён, лишнее потеряно, rewarded (сохранить 1 предмет).

---

### InventoryScene

Экипировка (3 слота) + запас (8–12). Тап → «Надеть» / «Продать» / «Починить».

---

### DevPanelScene

Ресурсы, билеты, снаряжение, действия (починить всё, сбросить, заполнить сундук, уровень).
