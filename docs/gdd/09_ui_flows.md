## Описание экранов

### MainMenuScene

**Назначение:** Точка входа в игру.

```
┌──────────────────────────────┐
│                              │
│         BIG HEROES           │  ← Заголовок, 42px Black
│                              │
│     ┌──────────────────┐     │
│     │                  │     │
│     │   [Герой/лого]   │     │
│     │                  │     │
│     └──────────────────┘     │
│                              │
│   ┌────────────────────────┐ │
│   │      ИГРАТЬ            │ │  ← ButtonPrimary (зелёный)
│   └────────────────────────┘ │
│   ┌────────────────────────┐ │
│   │    ПРОДОЛЖИТЬ          │ │  ← ButtonPrimary (если есть сохранение)
│   └────────────────────────┘ │
│                              │
│           [v 1.0]            │  ← Версия, мелкий текст
└──────────────────────────────┘
```

**Действия:**
- «Играть» → новая игра → HubScene.
- «Продолжить» → загрузка SaveManager → HubScene.

---

### HubScene

**Назначение:** Центральный хаб между PvE и PvP. Управление героем, ремонт, переход в контуры.

```
┌──────────────────────────────┐
│ [Gold: 1500]   [Gems: 80]   │  ← ResourceBar (верхняя панель)
├──────────────────────────────┤
│                              │
│    ┌──────────────────┐      │
│    │   [Аватар героя] │      │
│    │   Масса: 420     │      │  ← accent_cyan, крупно
│    │   Рейтинг: 1150  │      │
│    └──────────────────┘      │
│                              │
│ ┌─────────┐  ┌─────────┐    │
│ │ Weapon  │  │ Armor   │    │  ← Слоты экипировки (карточки)
│ │ Меч t1  │  │ Кольч.  │    │
│ │ 70/100  │  │ 45/100  │    │
│ └─────────┘  └─────────┘    │
│       ┌─────────┐            │
│       │ Access. │            │
│       │ Кольцо  │            │
│       │ 100/100 │            │
│       └─────────┘            │
│                              │
│ ┌──────────────────────────┐ │
│ │   ПОХОД (билеты: 3/5)   │ │  ← ButtonPrimary
│ └──────────────────────────┘ │
│ ┌──────────────────────────┐ │
│ │   АРЕНА (билеты: 2/3)   │ │  ← ButtonSecondary (accent_magenta)
│ └──────────────────────────┘ │
│                              │
├──────────────────────────────┤
│ [Герой] [Инвент.] [Ремонт] [Dev] │ ← BottomNav
└──────────────────────────────┘
```

**Действия:**
- Тап по слоту экипировки → InventoryScene (фильтр по слоту).
- «Поход» → экран подготовки PvE → PveMapScene.
- «Арена» → экран подготовки PvP → PvpLobbyScene.
- BottomNav: переключение вкладок.
- «Dev» → DevPanelScene (только в dev-сборке).

### PveMapScene

**Назначение:** Экран текущего похода. Показывает позицию и выбор пути.

```
┌──────────────────────────────┐
│ [Gold: 1500]   [HP: ██░ 75%] │
│ [Масса: 420]   [Узел 3/10]  │
├──────────────────────────────┤
│                              │
│   Текущий узел: РАЗВИЛКА     │
│   Выберите путь:             │
│                              │
│ ┌────────────┐ ┌────────────┐│
│ │  ⚔️ БОЙ    │ │  🎁 СУНДУК ││  ← Карточки узлов
│ │ Рядовой моб│ │ Добыча без ││
│ │            │ │ боя        ││
│ └────────────┘ └────────────┘│
│       ┌────────────┐         │
│       │  💀 ЭЛИТА   │         │
│       │ Сильный моб│         │
│       │ Лучше лут  │         │
│       └────────────┘         │
│                              │
│ ┌──────────────────────────┐ │
│ │      ОТСТУПИТЬ           │ │  ← ButtonDanger
│ └──────────────────────────┘ │
│ Рюкзак: [🗡️] [🛡️] [  ] [  ] │  ← Мини-рюкзак внизу
└──────────────────────────────┘
```

**Состояния:**
- Развилка: показать варианты путей (2–3 карточки).
- Линейный узел: показать одну карточку, кнопка «Продолжить».
- Бой: переход к BattleScene.
- Магазин/лагерь/событие: переход к ShopScene / CampScene / модальное окно события.

### BattleScene

**Назначение:** Экран боя (общий для PvE и PvP).

```
┌──────────────────────────────┐
│  ⚔️ БОЙ                      │
├──────────────────────────────┤
│                              │
│  ┌─────────┐  ┌─────────┐   │
│  │ [Герой] │  │ [Враг]  │   │
│  │         │  │         │   │  ← Два «яйца» / аватара
│  │ 420 гр  │  │ 350 гр  │   │
│  └─────────┘  └─────────┘   │
│                              │
│  HP героя: ████████░░ 80%   │  ← ProgressBar (зелёный)
│  HP врага: ██████░░░░ 60%   │  ← ProgressBar (красный)
│                              │
│  ┌──────────────────────┐    │
│  │ Лог боя:             │    │
│  │ Вы нанесли 45 урона  │    │  ← Лог последних 3 действий
│  │ Враг нанёс 22 урона  │    │
│  └──────────────────────┘    │
│                              │
│ ┌──────────────────────────┐ │
│ │      АТАКОВАТЬ           │ │  ← ButtonPrimary
│ └──────────────────────────┘ │
└──────────────────────────────┘
```

**Обратная связь:**
- Нажатие «Атаковать» → анимация удара (shake) + всплывающее число урона.
- Ответный удар врага → shake героя + красное число.
- Крит → увеличенное число + метка «КРИТ!».
- HP < 25% → полоса красная, пульсация.

### PvpResultScene

**Назначение:** Результат аренного боя.

```
┌──────────────────────────────┐
│       ПОБЕДА!                │  ← Ribbon (accent_pink) или
│  (или ПОРАЖЕНИЕ — красный)   │     (accent_red)

│                              │
│  Рейтинг: 1150 → 1158 (+8)  │

│  Прочность: −10 к каждому    │

│                              │

│ │     ПРОДОЛЖИТЬ           │ │  ← ButtonPrimary → PvpLobbyScene
│ └──────────────────────────┘ │
│ ┌──────────────────────────┐ │
│ │   ЗАВЕРШИТЬ СЕССИЮ       │ │  ← ButtonDanger → HubScene
```

### InventoryScene

**Назначение:** Управление снаряжением и инвентарём.

```
┌──────────────────────────────┐
│  ИНВЕНТАРЬ                   │
├──────────────────────────────┤
│  Экипировка:                 │
│  ┌────────┐┌────────┐┌────────┐
│  │Weapon  ││Armor   ││Access. │
│  │Меч t1  ││Кольч.  ││Кольцо │
│  │70/100  ││45/100  ││100/100│
│  └────────┘└────────┘└────────┘
│                              │
│  Запас:                      │
│  ┌────────┐┌────────┐┌────────┐
│  │Топор t1││Латы t1 ││Амулет │
│  │80/80   ││60/80   ││100/100│
│  └────────┘└────────┘└────────┘
│  ┌────────┐┌────────┐         │
│  │Оберег  ││ пусто  │         │
│  │100/100 ││        │         │
│  └────────┘└────────┘         │
│                              │
│ [Надеть] [Продать] [Починить]│  ← Действия (при выбранном предмете)
└──────────────────────────────┘
```

**Взаимодействие:**
- Тап по предмету → выделение + кнопки действий.
- «Надеть» → если слот занят, старый предмет → запас.
- «Продать» → подтверждение → Gold.
- «Починить» → показать стоимость → подтверждение → Gold списывается, durability восстановлена.

## Переходы между экранами

| Из | В | Триггер | Анимация |
|----|---|---------|----------|
| MainMenu | HubScene | Кнопка «Играть» / «Продолжить» | Fade in 300ms |
| HubScene | PveMapScene | Кнопка «Поход» + подтверждение | Slide left 300ms |
| HubScene | PvpLobbyScene | Кнопка «Арена» + подтверждение | Slide left 300ms |
| PveMapScene | BattleScene | Выбор узла боя | Fade 200ms |
| BattleScene | LootScene | Победа в PvE | Bounce 300ms |
| BattleScene | PvE Result | Поражение в PvE | Fade 300ms |
| LootScene | PveMapScene | «Взять» / «Пропустить» | Slide right 200ms |
| PveMapScene (boss) | HubScene | Победа над боссом | Fade 300ms через Victory |
| PvpLobbyScene | BattleScene | «Напасть» | Fade 200ms |
| BattleScene | PvpResultScene | Конец PvP-боя | Fade 200ms |
| PvpResultScene | PvpLobbyScene | «Продолжить» | Slide right 200ms |
| PvpResultScene | HubScene | «Завершить» | Slide right 300ms |
| Любой | DevPanelScene | Горячая клавиша / кнопка | Modal overlay |
