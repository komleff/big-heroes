## Мобы PvE — глава 1

Мобы имеют кастомные статы. HP = масса (как у игрока). Сила и броня задаются вручную для создания различных боевых профилей.

### Профили мобов

| Профиль | Характеристика | Пример |
|---------|---------------|--------|
| Слабый | Низкая масса, минимальная угроза | Слизень |
| Быстрый | Высокая сила относительно массы, низкая броня | Волк |
| Стандартный | Сила ≈ масса/3, умеренная броня | Гоблин |
| Бронированный | Сила ниже масса/3, высокая броня | Скелет |
| Стеклянная пушка | Высокая сила, низкие HP | Тёмный маг |
| Танк | Высокие HP и броня, умеренная сила | Огр |

## Pity-система лута

Каноническое описание: [13_boosters_relics](13_boosters_relics.md). Кратко: каждый 5-й предмет из сундуков и боёв в PvE гарантированно tier+1. С бустером meta_loot_pity — каждый 3-й.
