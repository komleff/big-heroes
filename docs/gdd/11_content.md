## Источники расходников

| Источник | Что выпадает |
|----------|-------------|
| PvE: combat | Аптечка t1, ремкомплект t1, факел, сухпаёк (шанс 15–25%) |
| PvE: elite | Зелья t1–2, яд t1, подзорная труба (гарантировано 1) |
| PvE: boss | Зелья t2–3, яд t2, особые (гарантировано 2) |
| PvE: chest | Аптечка t1–2, ремкомплект t1–2, факел, котомка (1–2 предмета) |
| PvE: shop | Любые доступные в текущей главе за Gold |
| Аренный сундук | Зелья t1–2, аптечка t2, ремкомплект t2 (1–2 предмета) |

## Pity-система лута

| Параметр | Значение |
|----------|----------|
| pity_counter | 5 |
| Эффект | Каждый 5-й предмет = гарантированный tier+1 |
| С бустером | pity_counter = 3 |
| Область | В рамках одного похода |
