# Big Heroes — Game Design Document

**Версия:** 1.1
**Дата:** 2026-04-04
**Статус:** Draft

---

## Краткое описание

Big Heroes — мобильная F2P-игра в жанре asynchronous hybrid RPG battler. Игрок проходит PvE-походы с экстракшн-механикой (ограниченный рюкзак, риск потери лута), затем использует добытое в асинхронных PvP-боях на арене. Два контура связаны единой прогрессией: PvE готовит к PvP, PvP мотивирует возвращаться в PvE.

## Цели дизайна

- Создать повторяемый цикл «подготовка → забег → конвертация результата → новый забег», где каждый заход ощущается осмысленным.
- Дать чувство контроля над риском и наградой через экстракшн-логику.
- Обеспечить устойчивую F2P-экономику через встроенные стоки (прочность, ремонт, потеря массы).
- Доказать жизнеспособность двойного цикла в MVP и довести до soft launch за 3 месяца.

## Жанровое позиционирование

RPG-extraction battler: «короткие PvE-рейды за лут → подготовка билда → арена за сундуки и ранги».

Дифференциация от конкурентов (RAID, Hero Wars, Summoners War, Archero):

| Механика | Big Heroes | Конкуренты |
|----------|-----------|------------|
| Экстракшн-логика (рюкзак, выбор, потери) | Да | Нет |
| Прочность снаряжения как системный сток | Да | Нет у большинства |
| Масса как универсальный ограничитель прогресса | Да | Нет (обычно power score без издержек) |

## Целевая аудитория

| Сегмент | Мотивация | Приоритет |
|---------|-----------|-----------|
| Midcore RPG оптимизаторы билда | Сбор мета-билда, доминирование в PvP | Очень высокий |
| Roguelite-раннеры (Archero-тип) | Короткие забеги, вариативность, рост силы | Высокий |
| Strategy/4X-гильдейщики | Социальная конкуренция, контроль ресурсов | Высокий (post-MVP) |
| Экстракшн-фанаты | Адреналин от риска потери, ценность лута | Средний (УТП) |
| Собиратели прогресса | Коллекции, ежедневные ритуалы | Средний |

## Scope

| Аспект | MVP | Post-MVP (6 мес.) |
|--------|-----|-------------------|
| Герои | 1 | 3–5 |
| PvE-главы | 1 (обычная) | 3–5 (до редкой/эпической) |
| PvP-лиги | 1 | 3–5 |
| Снаряжение | Tier 1, 3 слота, 9 предметов | Tier 1–3, расширенный пул |
| Экономика | Gold + Gems | + Season Pass, подписка |
| Социал | Нет | Кланы, рейды (скелет) |

## Глоссарий

| Термин | Определение |
|--------|-------------|
| Масса (mass) | Основной боевой ресурс героя. Растёт в PvE, теряется при поражении в PvP. Влияет на урон. |
| Прочность (durability) | Ресурс снаряжения. −1 при использовании команды предмета. При 0: оружие/щит = команда доступна без бонуса; аксессуар = команда заблокирована. |
| Аренный сундук (arena_chest) | Шкала прогресса в PvP. Заполняется за активные онлайн-бои. При заполнении — награда. |
| Рюкзак (backpack) | Ограниченное число слотов для выноса лута из PvE-похода. |
| Узел (node) | Точка на маршруте похода: бой, сундук, магазин, лагерь, событие. |
| Развилка (fork) | Точка маршрута, где игрок выбирает один из 2–3 путей. |
| Бот (bot) | Автоматически сгенерированный противник для PvP-арены. |
| Gold | Софт-валюта для ремонта, покупок в магазине. |
| Gems | Хард-валюта для расширения рюкзака, билетов, премиальных удобств. |
| Tier | Уровень/редкость предмета (1–5: обычный → легендарный). Каталог MVP: tier 1. Каталог полный: tier 1–4. |
| Бустер | Мета-усилитель фарма или PvP. Действует N боёв / M минут. Активируется в хабе. |
| Реликвия | Пассивный бонус, найденный в PvE-походе. Действует до конца похода, не переносится. |
| Камбек-бонус | Бустер, выдаваемый автоматически при возвращении или серии поражений. |
| Pity-система | Гарантия: каждый 5-й предмет из лута — tier+1. |
| Святилище (sanctuary) | Первый узел похода. Выбор 1 из 3 реликвий. |
| Пояс (belt) | 3–4 слота для расходников. Доступен перед боем. |
| Билет кампании (campaign_ticket) | Расходуется при входе в PvE-поход. |
| Билет арены (arena_ticket) | Расходуется при входе на PvP-арену. |
