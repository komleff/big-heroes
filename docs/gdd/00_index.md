## Краткое описание

Big Heroes — мобильная F2P-игра в жанре asynchronous hybrid RPG battler. Игрок проходит PvE-походы с экстракшн-механикой (ограниченный рюкзак, риск потери лута), затем использует добытое в асинхронных PvP-боях на арене. Два контура связаны единой прогрессией: PvE готовит к PvP, PvP мотивирует возвращаться в PvE.

## Цели дизайна

- Создать повторяемый цикл «подготовка → забег → конвертация результата → новый забег», где каждый заход ощущается осмысленным.
- Дать чувство контроля над риском и наградой через экстракшн-логику.
- Обеспечить устойчивую F2P-экономику через встроенные стоки (прочность, ремонт, потеря массы).
- Доказать жизнеспособность двойного цикла в MVP и довести до soft launch за 3 месяца.

## Жанровое позиционирование

RPG-extraction battler: «короткие PvE-рейды за лут → подготовка билда → арена за сундуки и ранги».

| Механика | Big Heroes | Конкуренты |
|----------|-----------|------------|
| Экстракшн-логика (рюкзак, выбор, потери) | Да | Нет |
| Прочность снаряжения как системный сток | Да | Нет у большинства |
| Масса как универсальный ограничитель прогресса | Да | Нет (обычно power score без издержек) |

## Решения v1.2

### R1. Масса при поражении/выходе из PvE

| Условие | Масса |
|---------|-------|
| Победа над боссом | Вся набранная за поход сохраняется |
| Поражение в бою | Масса на момент входа в проигранный бой |
| Добровольный выход | Вся набранная сохраняется |

### R2. HP в бою — единая модель

HP = масса, полное восстановление перед каждым боем. Одинаково в PvE и PvP. Бой — чистое решение «команда + расходник vs враг». Прочность снаряжения — единственный ресурс на истощение в походе. Аптечки удалены. Целебные реликвии заменены. Лагерь = ремонт/тренировка.

### R3. cmd_poison → cmd_polymorph

acc_vial_t1 «Колба алхимика» → cmd_polymorph.

### R4. Статы мобов — кастомные профили

Мобы имеют кастомные статы для разных боевых профилей. HP = масса, сила и броня — вручную.

### R5. Полиморф в PvP

0 рейтинга, 0 сундука, масса без изменений. Аксессуар −1 прочность.

### R6. Гейты разблокировки глав

Три условия: босс + лига + уровень аккаунта.

### R7. Пояс на уровне 1

Пояс недоступен до уровня аккаунта 2.

## Глоссарий

| Термин | Определение |
|--------|-------------|
| Масса (mass) | Основной боевой ресурс героя. HP = масса (1:1). Сила = масса/3 + бонусы. Растёт в PvE, теряется при поражении в PvP (−10%). |
| Прочность (durability) | Ресурс снаряжения. −1 при использовании команды предмета. При 0: оружие/щит = команда без бонуса; аксессуар = команда заблокирована. Единственный ресурс на истощение в походе. |
| Аренный сундук (arena_chest) | Шкала прогресса в PvP (10 делений). Заполняется за онлайн-бои. |
| Рюкзак (backpack) | Ограниченное число слотов для выноса лута из PvE-похода. Базовый: 4, максимум: 8. |
| Узел (node) | Точка на маршруте похода: бой, сундук, магазин, лагерь, событие. |
| Развилка (fork) | Точка маршрута, где игрок выбирает один из 2–3 путей. |
| Бот (bot) | Автоматически сгенерированный противник для PvP-арены. |
| Gold | Софт-валюта для ремонта, покупок в магазине. |
| Gems | Хард-валюта для расширения рюкзака, билетов, премиальных удобств. |
| Tier | Уровень/редкость предмета (1–4: обычный → эпический). MVP: tier 1. |
| Бустер (booster) | Мета-усилитель фарма или PvP. Действует N боёв + M минут. Активируется в хабе. |
| Реликвия (relic) | Пассивный бонус, найденный в PvE-походе. Действует до конца похода, максимум 3. |
| Камбек-бонус (comeback) | Бустер, выдаваемый автоматически при возвращении или серии поражений. |
| Pity-система | Гарантия: каждый 5-й предмет из лута — tier+1 (с бустером — каждый 3-й). |
| Святилище (sanctuary) | Первый узел похода. Выбор 1 из 3 реликвий. |
| Пояс (belt) | 2–4 слота для расходников. Разблокируется на уровне аккаунта 2. |
| Билет кампании (campaign_ticket) | Расходуется при входе в PvE-поход. |
| Билет арены (arena_ticket) | Расходуется при входе на PvP-арену. |
| Путь аккаунта (account_path) | Перманентная система разблокировки фич через XP аккаунта. |
| Уровень героя (hero_level) | Перманентная прокачка героя через XP боёв. Даёт +массу и пассивное умение. |
| Снапшот (snapshot) | Копия героя, сохраняемая после PvP-победы для оффлайн-защиты. |
| АФК-награда (afk_reward) | Ресурсы за время отсутствия (с уровня аккаунта 15). |
