# Универсальные правила

## Язык
- Комментарии в коде — только на русском
- Идентификаторы — на английском

## Git workflow
- Пуш в main запрещён — только ветки и PR
- Merge — только оператор
- Checkpoint-коммиты перед опасными изменениями

## Качество
- После изменений: `npm run build` && `npm run test`
- Атомарность: одна задача = один фокус
- Работай по задачам — не делай ничего, что не запрошено
- Не "улучшай" стиль, архитектуру, форматирование без запроса

## Эскалация
- При расхождении требований — сообщить оператору
- При конфликте источников правды — сообщить оператору
