# Правила для клиентского кода

> Архитектурные паттерны — см. `.memory_bank/systemPatterns.md`

## Тактические правила

- PixiJS v8 + TypeScript
- Точка входа: `client/src/main.ts` — создаёт `Application`, ничего больше
- Сцены — отдельные классы, наследуют от `Container`
- Переходы между сценами — через `SceneManager`
- Формулы и бизнес-логика — только из `shared/`; клиент не считает сам
- Ассеты загружаются через `Assets.load()` — суммарно до 10 МБ
- Не добавляй тяжёлые зависимости без согласования

## Структура `client/src/`

```
main.ts          — точка входа
scenes/          — классы сцен (один файл = одна сцена)
ui/              — HUD, кнопки, панели (Container-компоненты)
assets/          — манифест ассетов
utils/           — вспомогательные функции
```

## Запрещено

- Прямые манипуляции с DOM (кроме `app.canvas`)
- Импорт из `server/` (пакета нет)
- Бизнес-расчёты вне `shared/`
- Хардкод числовых констант вне `config/balance.json`
