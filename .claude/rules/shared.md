# Правила для shared-пакета

## Назначение
`shared/src/` — единственный источник игровой математики и типов-контрактов.
Клиент импортирует формулы и типы отсюда.

## Чистые функции
- Все функции — чистые: детерминированный результат, без side-effects
- Нет обращений к сети, файловой системе
- Нет мутации входных аргументов

## Зависимости
- Shared-пакет не зависит от `client/`
- Минимум внешних зависимостей (в идеале — ноль)
- Никаких Node.js-специфичных API (должен работать в браузере)

## Тестируемость
- Каждая формула покрыта юнит-тестами
- Тесты проверяют граничные значения и бизнес-правила из `config/balance.json`

## Запрещено
- Импортировать из `client/`
- Использовать side-effects (console.log в продакшн-коде, fetch, fs)
- Менять контракты без обновления архитектурного документа
- Добавлять зависимости от Node.js API
