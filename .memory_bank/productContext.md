# Продуктовый контекст — Big Heroes

**Обновлён:** 2026-04-02

---

## Описание проекта

Big Heroes — мобильная F2P asynchronous hybrid RPG battler на PixiJS v8. Два связанных контура прогрессии: PvE-кампания (рогалик-поход с extraction-механикой) и PvP-арена (асинхронные бои с ботами). Сингл-клиент, без сервера.

**Продуктовый тезис:** игра про подготовку, серию решений и конвертацию подготовки в аренный результат.

## Платформа

- Standalone браузер (без серверной части в демо/прototipe)
- Целевая платформа: мобильный браузер / WebGL

## Целевая аудитория

Игроки мобильных midcore RPG-баттлеров. Знакомы с петлёй «копи → бейся → получи награду». Ценят контроль над риском и ощущение прогресса.

## Главный цикл

```
PvE-поход → добыча (масса, снаряжение, бустеры, Gold) → вынос лута (рюкзак)
→ экипировка → PvP-арена → рейтинг + износ + масса → аренный сундук
→ награды (PvE-бустеры, осколки) → новый поход
```

## Ключевые механики MVP

**PvE-поход:**
- 1 глава, 8–10 узлов; типы: бой, элита, магазин, лагерь, событие, сундук, босс
- Линейное движение с развилками (выбор из 2–3 путей)
- Вход: 1 campaign_ticket

**Боевая система:**
- Пошаговый бой 1-на-1 (единая система для PvE и PvP)
- Урон: `base_mass × equipment_modifier × booster_modifier`
- HP: отдельный показатель; полностью восстанавливается между походами

**Ресурсы:** mass, gold, gems, campaign_ticket, arena_ticket, pve_booster, pvp_booster, chest_progress, collection_item

**Инвентарь:**
- 3 слота экипировки (weapon, armor, accessory)
- Рюкзак: 4 слота в походе (платное расширение до 8)
- Запас: 8–12 слотов
- Прочность: изнашивается в PvP; ремонт за Gold

**PvP-арена:**
- 1 лига, боты; вход: 1 arena_ticket + mass ≥ min_arena_mass лиги
- Победа: +рейтинг, +chest_progress; Поражение: −рейтинг, −mass
- Аренный сундук: прогресс только за активные онлайн-сессии

**Прогрессия:**
- 5 глав PvE (tier 1–5) + 5 лиг PvP; разблокировка требует успеха в обоих контурах
- MVP: глава 1 + лига 1

**Монетизация:** IAP + IAA; донат = удобство и скорость, не прямая сила

**Dev-панель:** Gold/Gems/масса/рейтинг/снаряжение/бустеры — обязательна для QA

## KPI-цели MVP (из GDD)

- D1 retention ≥ 30%
- Цикл жизнеспособности доказан раньше soft launch

## Роадмап (из GDD)

- Месяц 1: Прototip core feel — PvE-забег с боями, лутом, рюкзаком
- Месяц 2: Полный MVP — PvP, экономика, монетизация, DevPanel
- Месяц 3: Полировка, soft launch
